## 📘 What it is?

A complete learning resource for **Python fundamentals applied to Data Science** — starting from programming essentials and moving through:  
- Iterators and generators  
- SQL with **SQLite**  
- NoSQL with **MongoDB**  
- Core data libraries: **pandas** and **NumPy**  

This project demonstrates how to design **ETL pipelines** and perform **Exploratory Data Analysis (EDA)** using real-world techniques and datasets.

---

## 🎯 Why I built this project?

This repository was created to help **students and professionals** master the tools needed for data manipulation and analysis.  
It connects **Python programming** with **database integration** and **data transformation workflows**, offering an end-to-end perspective from raw data to insights.

## ⚙️ Getting Started  

Follow these steps to set up your local environment:

```bash
git clone https://github.com/jorgeav527/py-fundamentals-ds.git
cd py-fundamentals-ds
python3 -m venv .venv
source .venv/bin/activate   # On Linux/Mac
.\venv\Scripts\activate      # On Windows
pip install -r requirements.txt
jupyter lab                # Or jupyter-lab
```
